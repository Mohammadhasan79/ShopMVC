# 🛒 ShopMVC - ASP.NET MVC E-Commerce Project

## 📌 Overview
ShopMVC is a simple e-commerce web application built with ASP.NET MVC.  
It demonstrates core concepts of MVC architecture, database integration, and basic online shopping functionality.

---

## 🛠 Technologies Used
- ASP.NET MVC
- C#
- Entity Framework
- SQL Server
- HTML / CSS / Bootstrap
- JavaScript / jQuery

---

## 🏗 Architecture
This project follows the MVC (Model-View-Controller) pattern:

- **Model:** Data structure and business logic
- **View:** User interface (Razor Views)
- **Controller:** Handles user requests and application flow

---

## 🚀 Features
- Product Listing
- Product Details Page
- Category Management
- Shopping Cart
- Basic Order System
- Admin/Product Management (if exists)

---

## 🗄 Database
- SQL Server used for data storage
- Entity Framework used for ORM

---

## 📂 Project Structure
- Models
- Views
- Controllers
- Data Access Layer (if exists)

## 🎯 Purpose
This project was developed for learning and practicing ASP.NET MVC architecture and building real-world web applications.
